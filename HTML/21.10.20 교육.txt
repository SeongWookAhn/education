입력양식

form
홈페이지에서 사용자에게 정보를 입력받을때 사용하는 사용자 인터페이스
기본적인 텍스트 입력, 선택, 버튼 파일첨부 가능
버튼의 경우 자바스크립트와 연동
파일첨부는 서버에서 파일업로드를 별도로 구현되어있어야함
form 태그를 통해 작성된 입력양식은 최종적으로 서버에 전송
주속성은 method와 action이다.

Get 방식

데이터를 요청할때 사용
url길이 제한 있음
중요한 작업을 할때에는 get 방법을 사용하면 안됨(url에 정보가 표기됨)

Post 방식

회원가입이나 게시판에 글을 쓰는 것 같은 데이터를 서버에 전송할때 사용

전송할 폼 데이터에 중요정보나 개인정보가 포험되어 있으면 항상 POST방식 사용권장.
제출된 양식 데이터를 페이지 주소필드에 표시하지 않음.
POST 방식에는 크기 제한이 없으므로 많은 양의 데이터를 보낼수있음.
파일전송은 multipart/form-data 로 전송해야 하며 일반적인 텍스트 데이터 전송과는 다름.

기본적으로는 <input> 태그를 사용하며 type속성으로 다양한 입력 양식을 정의할 수 있다.
그외 <select>, <textarea>, <button> 등의 태그도 사용할 수 있다.

ㄴ input
가장 많이 사용하는 입력 양식 태그이며 대부분의 입력 기능을 제공
Type속성으로 입력 양식의 종류를 나타내고 name속성으로 서버에 전송될 데이터
이름을 지정 Value 속성은 기본 입력값을 지정할 때 사용하고 placeholder는 입력 항목에
대한 설명문을 넣을때 사용한다.

Type

필수 속성이며 사용할수있는 type은 다음과 같다.
text : 텍스트 입력 필드
password : 비밀번호 입력 필드
checkbox : 복수 선택 가능 체크박스
radio : 복수 선택 불가능한 라디오 버튼 생성
submit : 입력 양식을 서버로 전송하기 위한 버튼
reset : 리셋버튼
button : 일반 버튼

※placeholder : 박스안에 텍스트를 표시
